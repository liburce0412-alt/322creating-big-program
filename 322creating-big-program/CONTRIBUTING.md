# Contributing

Thanks for your interest in CampusAI!

## How to Contribute

1. Fork the repository
2. Create a feature branch (git checkout -b feature/amazing)
3. Commit your changes (git commit -m "Add amazing feature")
4. Push to the branch (git push origin feature/amazing)
5. Open a Pull Request

## Guidelines

- Keep code style consistent with existing files
- Add smoke tests for new API endpoints
- Frontend changes should preserve the warm terracotta design system
- C module changes need Python fallback (c_bridge.py)
- Run python3 tests/smoke_test.py before submitting
